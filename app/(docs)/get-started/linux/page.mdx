## API Reference

### Button

- `variant` (string): The style variant for the button. Options include `primary`, `secondary`, and `tertiary`.
- `onClick` (function): Callback function to handle button clicks.

### Modal

- `isOpen` (boolean): Determines if the modal is displayed.
- `onClose` (function): Callback fired when the modal is closed.
