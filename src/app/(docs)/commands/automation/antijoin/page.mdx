# `antijoin`

Turns anti join on or off. Makes it so that if someone joins while it's activated, they get kicked from the server.

## Notes

-raidmode isn't a slash command. Only a text command.

### Aliases
`raidmode`

### Syntax

```
-antijoin
-raidmode
/antijoin
```

### Required Permissions

`KickMembers`
