# `gafk`

Sets your AFK status, in all servers the bot is in.
When you come back, it will tell you if anyone pinged you, and if someone did, it will link you to the message.

### Aliases
`gafk`

### Syntax

```
-gafk [reason]
```

### Required Permissions

`NONE`

### Modes

<Modes modes={["legacy"]} />

### Examples

```
-gafk Sleeping
-gafk Working
```
