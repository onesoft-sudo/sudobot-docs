# `queue cancel`

Cancels a command queue.

### Aliases
`queue remove`
`queuecancel`
`cancelqueue`
`rmqueue`
`delqueue`
`removequeue`
`queuedel`

### Syntax

```
-queue cancel [queue ID]
-queue__remove [queue ID]
-queuecancel [queue ID]
-cancelqueue [queue ID]
-rmqueue [queue ID]
-delqueue [queue ID]
-removequeue [queue ID]
-queuedel [queue ID]
/queue cancel id: [queue ID]
```

### Required Permissions

`ManageMessages`

### Modes

<Modes modes={["legacy", "interaction"]} />

### Examples

```
-queue cancel 34
-queue remove 183
-queuecancel 3
-cancelqueue 623
-delqueue 187
-removequeue 54
-queuedel 654
/queue cancel id: 84
```
